# Cinematic Portfolio Showcase
## Jesnow Biju | Video Editor & Graphic Designer

A high-end, Netflix-style cinematic portfolio built with HTML5, CSS3, and JavaScript. Featuring a 3D animated hero background, professional horizontal video galleries, and interactive fullscreen playback.

### 🎥 Key Features
- **Cinematic Showcase**: 4-row organized gallery with "one-click" fullscreen playback.
- **3D Hero Visuals**: Floating Adobe and design software icons with realistic depth and flow.
- **Creative Typography**: Electric Cyan and Neon Magenta branding.
- **Glassmorphism UI**: Modern, boutique interface with high-performance video streaming.

### 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Style**: Vanilla CSS3 (Custom animations)
- **Logic**: Vanilla JavaScript (ES6+)
- **Hosting**: Vercel

### 🚀 Deploying to Vercel
1. Upload this repository to GitHub.
2. Link the repository to your Vercel account at [vercel.com](https://vercel.com).
3. The site will deploy automatically to **jesnowbiju.vercel.app**.
